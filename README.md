# Registration-System

A Java **Course Registration System** with MySQL database integration.

## Structure

```
Registration-System/
├── Main.java            # Entry point
├── Admin.java           # Admin role management
├── Student.java         # Student operations
├── Faculty.java         # Faculty operations
├── Courses.java         # Course management
├── Schedule.java        # Scheduling logic
├── DBConnection.java    # MySQL connection handler
├── DBConfig.java        # DB configuration
├── Common_function.java # Shared utilities
└── ScannerUtil.java     # Input handling
```

## Features
- Role-based access: Admin, Student, Faculty
- Course registration and scheduling
- MySQL database backend

## Requirements
Java 11+, MySQL, mysql-connector-j
